# Url shortener web application.

**To run this web application locally, you need to have docker and 1.8 GB of free space.**

In the root directory, run the following command:
```
docker compose up --build
```
After that, you can use this application by url http://localhost:5000. There are two users: admin (email: admin@email.com; password: adminPass) and simple user (email: user@email.com; password: userPass). You can use them to log in.
